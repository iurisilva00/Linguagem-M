# Linguagem-M
## Nesse projeto utilizamos a linguagem m do power bi, para obter informações de uma API e transformar em tabela. 
## Comandos utilizados:
- each
- Funções anônimas
- List.Generate
- Table
- Table.TransformColumns
- Table.ExpandRecordColumn
- Table.ExpandListColumn
- Text.FromBinary
- Text.ToBinary
- TextEncoding.Utf8
- List.Accumulate
- List.Select
